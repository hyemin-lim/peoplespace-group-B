## 사용한 git 명령어

$git checkout frontEnd

$git checkout -b [feature/기능 요약] frontEnd : frontEnd에서 브랜치 만드는 것, [feature/기능 요약]은 

branch 이름 예시

 여기서 작업 후 나중에 frontEnd 브랜치로 merge 예정~~

                  - (가은님branch)
                
                  - (정은님branch)
